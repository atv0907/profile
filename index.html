Passive Vulnerability Management System
Capstone Project – Sheridan College

Designed and developed a passive vulnerability detection and management platform that identifies software vulnerabilities without the need for resource-heavy, disruptive active scans. The system imports software inventory data from Windows and Linux machines, then uses advanced string matching algorithms (Ratcliff/Obershelp) to accurately map software to Common Platform Enumeration (CPE) identifiers.

Key Features
High-Accuracy Matching Achieved 100% accuracy in controlled testing with known vulnerable software such as Google Chrome, Apache Log4j, MySQL, and Adobe Flash Player.
Lightweight Operation Eliminates the performance impact of active vulnerability scanning, making it ideal for large-scale or sensitive environments.
Scalable Architecture Supports importing inventory data from multiple systems and can be extended to integrate with ticketing systems like JIRA and real-time alerting tools.
Self Hosted Compliance Automation
Client Project – Confidential Client

Designed, developed and deployed automated workflows for collecting compliance data for SMB (small and medium businesses) to assist with audit preparation. The application leverages scheduled API calls, webhooks and custom scripts to either pull compliance data from third party platforms or execute compliance routines on cloud instances. The application is deployed and maintained using n8n and Azure.

Key Features
Automated Data Aggregation Scheduled and triggered routines to grab compliance data such as User, VM and Container Data from Azure using Microsoft Graph API. Custom scripts to pull asset inventory for on-prem hardware. Pulling data from third party platforms like BambooHR for employee auditing.
AI Content Revision Pushed data through AI agent pipelines to read and verify data and look for non-compliance. AI models returned findings as a summarized report delivered via email.
